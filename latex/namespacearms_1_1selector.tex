\doxysubsection{Detailed Description}
This namespace deals with setting up and displaying the autonomous selector. Almost all of it\textquotesingle{}s variables and functions are private, with the exception being the {\itshape auton} variables.

The arms selector will handle everything in the back end, so all you need to worry about is which autonomous is selected. Below is a recommended format to use the ARMS autonomous selector used by Purdue SIGBots (BLRS).

This int stores the currently selected autonomous.~\newline
The value will be positive if you are using the red side, and negative if you are using the blue side.~\newline
The value will be 0 if you are using the skills autonomous.~\newline
