Moves the chassis to a target point. Almost all parameters are optional. Technically, only the {\itshape target} parameter is required. However, it is recommended to provide the max parameter aswell so that you have control over the maximum speed of the chassis.

The {\itshape target} parameter is a vector of doubles that represents the target point (x, y), or pose (x, y, theta). For a target point, our standard point-\/point odometry motion is used. For a target pose, our Boomerang controller is used. More information on these can be seen at Motion\+Control.~\newline
 The {\itshape max} parameter can be used to set the maximum speed of the movement. It will default to 100\% if not provided.~\newline
The {\itshape exit\+\_\+error} parameter can be used to set the minimum error from the target point to exit the movement. It will default to what is provided in \mbox{\hyperlink{config_8h_a80c2c0cbab028b3b18d7c7c521815721}{config\+::h\+::\+MIN\+\_\+\+ERROR}} if not provided.~\newline
The {\itshape lp} parameter can be used to set the linear kP for the movement. It will default to what is provided in \mbox{\hyperlink{config_8h}{config.\+h}} if not provided.~\newline
The {\itshape ap} parameter can be used to set the angular kP for the movement. It will default to what is provided in \mbox{\hyperlink{config_8h}{config.\+h}} if not provided.~\newline
The {\itshape flags} parameter can be used to set the flags for the movement. A list of them and their descriptions can be found in the \mbox{\hyperlink{structarms_1_1_move_flags}{Move\+Flags}} enum.~\newline
