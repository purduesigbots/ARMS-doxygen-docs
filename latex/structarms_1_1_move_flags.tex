\doxysubsection{Detailed Description}
This struct contains the flags that are used to control the movement of the robot.~\newline
There are 4 different flags for this struct, each of which is a boolean value.~\newline
 The {\itshape ASYNC} flag will cause the movement to be asynchronous, meaning that the robot will not wait for the movement to finish before continuing the code.~\newline
The {\itshape RELATIVE} flag will cause the movement to be relative to the robot\textquotesingle{}s current position, instead of global odom coordinates.~\newline
The {\itshape THRU} flag will disable PID control for the movement, causing the robot to move at maximum speed for that movement.~\newline
The {\itshape REVERSE} flag will cause the robot to move towards the movement using it\textquotesingle{}s \char`\"{}back\char`\"{} instead of it\textquotesingle{}s \char`\"{}front\char`\"{}. If you tell the robot to move to a point behind it without this flag, it will turn around and move to the point instead of moving backwards.~\newline
